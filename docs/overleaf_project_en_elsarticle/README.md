# overleaf_project_en_elsarticle

This is the **English-only** Elsevier/ScienceDirect (elsarticle) LaTeX project.
It now closely mirrors the official template you downloaded from Overleaf.

- Main file: `main.tex`
- Sections (requested structure):
	- `sections/01_introduction.tex`
	- `sections/02_related_work.tex`
	- `sections/03_problem_formulation.tex`
	- `sections/04_system_design.tex`
	- `sections/05_report.tex`
	- `sections/06_future_work.tex`
	- `sections/07_appendix.tex`
- BibTeX: `refs/references.bib`
- Bibliography style: `elsarticle-num.bst` (copied locally)

Notes:
- This folder is intentionally **isolated** from `docs/overleaf_project` (Chinese/IEEEtran version).
- Overleaf usually provides `elsarticle.cls` by default. For full parity with the downloaded package, this repo now also contains `elsarticle.cls` from the official bundle.
\n+- Compiler: **PDFLaTeX** (matches the official ScienceDirect template instructions)
